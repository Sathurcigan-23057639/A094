# FIFA 2018 Man of the Match Prediction

This project predicts the FIFA 2018 "Man of the Match" using statistical models.

## Files in the Repository
- `FIFA 2018 Statistics.csv`: Contains match data.
- `temp.R`: Code for the prediction model.
- `test 01.txt` and `test 02.txt`: Temporary test files.
- `.gitignore`: Files and directories excluded from version control.

## Usage
1. Load the `FIFA 2018 Statistics.csv` file.
2. Run the `temp.R` script in R to generate predictions.
